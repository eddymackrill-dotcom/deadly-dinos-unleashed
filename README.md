# Deadly Dinos Unleashed

A 2.5D side-scrolling action-adventure game inspired by BBC's *Deadly Mission Unleashed*. Play 8 prehistoric predators across themed missions — from the tiny Triassic Eoraptor to the Cretaceous king Spinosaurus.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

Three.js · Rapier · TypeScript · Vite · React (UI overlay) · Howler · GSAP · Zustand

## The Roster

1. **Eoraptor** — Late Triassic, Argentina
2. **Herrerasaurus** — Late Triassic, Argentina
3. **Deinonychus** — Early Cretaceous, North America
4. **Tarbosaurus** — Late Cretaceous, Mongolia
5. **Albertosaurus** — Late Cretaceous, Canada
6. **T-Rex** — Late Cretaceous, North America
7. **Carcharodontosaurus** — Mid Cretaceous, North Africa
8. **Spinosaurus** — Mid Cretaceous, North Africa
